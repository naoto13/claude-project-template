## 設定

### データストリーム

- プラットフォームごとに1ストリーム（ウェブ、iOS、Android）
- 自動トラッキングのために拡張計測機能を有効にする
- データ保持期間を設定する（デフォルト2ヶ月、最大14ヶ月）
- Google Signalsを有効にする（クロスデバイス用、同意取得が必要）

### 拡張計測イベント（自動）

| イベント | 説明 | 設定 |
|-------|-------------|---------------|
| page_view | ページ読み込み | 自動 |
| scroll | 90%スクロール深度 | オン/オフ切り替え可 |
| outbound_click | 外部ドメインへのクリック | 自動 |
| site_search | 使用された検索クエリ | パラメータ設定が必要 |
| video_engagement | YouTube動画再生 | オン/オフ切り替え可 |
| file_download | PDF、ドキュメントなど | 拡張子を設定可能 |

### 推奨イベント

拡張レポートのためにGoogleの定義済みイベントを可能な限り使用する:

**すべてのプロパティ共通:**
- login, sign_up
- share
- search

**Eコマース:**
- view_item, view_item_list
- add_to_cart, remove_from_cart
- begin_checkout
- add_payment_info
- purchase, refund

**ゲーム:**
- level_up, unlock_achievement
- post_score, spend_virtual_currency

参考: https://support.google.com/analytics/answer/9267735

## カスタムイベント

### gtag.js実装

```javascript
// 基本イベント
gtag('event', 'signup_completed', {
  'method': 'email',
  'plan': 'free'
});

// 値付きイベント
gtag('event', 'purchase', {
  'transaction_id': 'T12345',
  'value': 99.99,
  'currency': 'USD',
  'items': [{
    'item_id': 'SKU123',
    'item_name': 'Product Name',
    'price': 99.99
  }]
});

// ユーザープロパティ
gtag('set', 'user_properties', {
  'user_type': 'premium',
  'plan_name': 'pro'
});

// ユーザーID（ログイン済みユーザー向け）
gtag('config', 'GA_MEASUREMENT_ID', {
  'user_id': 'USER_ID'
});
```

### Google Tag Manager（dataLayer）

```javascript
// カスタムイベント
dataLayer.push({
  'event': 'signup_completed',
  'method': 'email',
  'plan': 'free'
});

// ユーザープロパティの設定
dataLayer.push({
  'user_id': '12345',
  'user_type': 'premium'
});

// Eコマース購入
dataLayer.push({
  'event': 'purchase',
  'ecommerce': {
    'transaction_id': 'T12345',
    'value': 99.99,
    'currency': 'USD',
    'items': [{
      'item_id': 'SKU123',
      'item_name': 'Product Name',
      'price': 99.99,
      'quantity': 1
    }]
  }
});

// 送信前にecommerceをクリアする（ベストプラクティス）
dataLayer.push({ ecommerce: null });
dataLayer.push({
  'event': 'view_item',
  'ecommerce': {
    // ...
  }
});
```

## コンバージョン設定

### コンバージョン作成

1. **イベントを収集する** — GA4でイベントが発火していることを確認する
2. **コンバージョンとしてマークする** — 管理 > イベント > コンバージョンとしてマーク
3. **カウント方法を設定する**:
   - セッションごとに1回（リード、サインアップ）
   - イベントごと（購入）
4. **Google Adsにインポートする** — コンバージョン最適化の入札のため

### コンバージョン値

```javascript
// コンバージョン値付きイベント
gtag('event', 'purchase', {
  'value': 99.99,
  'currency': 'USD'
});
```

コンバージョンをマークする際にGA4管理画面でデフォルト値を設定することもできる。

## カスタムディメンションとカスタム指標

### 使い分け

**カスタムディメンション:**
- セグメント・フィルターで使いたいプロパティ
- ユーザー属性（プランタイプ、業種）
- コンテンツ属性（著者、カテゴリ）

**カスタム指標:**
- 集計したい数値
- スコア、カウント、期間

### 設定手順

1. 管理 > データ表示 > カスタム定義
2. ディメンションまたは指標を作成する
3. スコープを選択する:
   - **イベント**: イベントごと（content_type）
   - **ユーザー**: ユーザーごと（account_type）
   - **アイテム**: 商品ごと（product_category）
4. パラメータ名を入力する（イベントパラメータと一致させる）

### 例

| ディメンション | スコープ | パラメータ | 説明 |
|-----------|-------|-----------|-------------|
| ユーザータイプ | ユーザー | user_type | 無料、トライアル、有料 |
| コンテンツ著者 | イベント | author | ブログ投稿の著者 |
| 商品カテゴリ | アイテム | item_category | Eコマースカテゴリ |

## オーディエンス

### オーディエンス作成

管理 > データ表示 > オーディエンス

**ユースケース:**
- リマーケティングオーディエンス（Adsにエクスポート）
- セグメント分析
- トリガーベースのイベント

### オーディエンスの例

**高意向ビジター:**
- 料金ページを閲覧した
- コンバージョンしていない
- 過去7日以内

**エンゲージドユーザー:**
- 3回以上のセッション
- または合計5分以上のエンゲージメント

**購入者:**
- 購入イベント
- 除外またはルックアライク用

## デバッグ

### DebugView

以下の方法で有効にする:
- URLパラメータ: `?debug_mode=true`
- Chrome拡張機能: GA Debugger
- gtag: configに`'debug_mode': true`を追加

確認場所: レポート > 設定 > DebugView

### リアルタイムレポート

30分以内のイベントを確認する:
レポート > リアルタイム

### よくある問題

**イベントが表示されない:**
- まずDebugViewを確認する
- gtag/GTMの発火を確認する
- フィルタ除外を確認する

**パラメータ値がない:**
- カスタムディメンションが未作成
- パラメータ名が一致していない
- データ処理中（24〜48時間かかる）

**コンバージョンが記録されない:**
- イベントがコンバージョンとしてマークされていない
- イベント名が一致していない
- カウント方法（1回 vs 毎回）を確認する

## データ品質

### フィルタ

管理 > データストリーム > [ストリーム] > タグ設定を構成 > 内部トラフィックを定義

**除外するもの:**
- 内部IPアドレス
- 開発者トラフィック
- テスト環境

### クロスドメイントラッキング

アナリティクスを共有する複数ドメインの場合:

1. 管理 > データストリーム > [ストリーム] > タグ設定を構成
2. ドメインを設定する
3. セッションを共有するすべてのドメインをリストアップする

### セッション設定

管理 > データストリーム > [ストリーム] > タグ設定を構成

- セッションタイムアウト（デフォルト30分）
- エンゲージドセッション時間（デフォルト10秒）

## Google Adsとの連携

### リンク設定

1. 管理 > プロダクトリンク > Google Adsのリンク
2. Google Adsで自動タグ設定を有効にする
3. Google Adsでコンバージョンをインポートする

### オーディエンスエクスポート

GA4で作成したオーディエンスはGoogle Adsで以下に使用できる:
- リマーケティングキャンペーン
- カスタマーマッチ
- 類似オーディエンス
