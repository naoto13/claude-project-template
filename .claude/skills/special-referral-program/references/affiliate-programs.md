## コミッション構造

**売上の割合:**
- 標準: 初回売上または初年度の10〜30%
- 適した用途: ECサイト、明確な価格設定のあるSaaS
- 例: 「紹介した売上ごとに25%を獲得」

**アクションごとの定額料金:**
- 標準: 価値に応じて5〜500ドル
- 適した用途: リード獲得、トライアル、フリーミアム
- 例: 「qualified demoごとに50ドル」

**継続コミッション:**
- 標準: 継続収益の10〜25%
- 適した用途: サブスクリプション製品
- 例: 「12ヶ月間、サブスクリプションの20%」

**段階コミッション:**
- 適した用途: ハイパフォーマーへのモチベーション付け
- 例: 「1〜10件は20%、11〜25件は25%、26件以上は30%」

## Cookie有効期間

クリック後、アフィリエイトはどのくらいの期間クレジットを受け取れるか？

| 期間 | ユースケース |
|----------|----------|
| 24時間 | 高ボリューム、低検討購買 |
| 7〜14日 | 標準的なECサイト |
| 30日 | 標準的なSaaS・B2B |
| 60〜90日 | 長いセールスサイクル、エンタープライズ |
| 無期限 | プレミアムアフィリエイト関係 |

## アフィリエイトの採用

### アフィリエイトを見つける場所:
- コンテンツを作成する既存顧客
- 業界のブロガーやレビュアー
- ニッチなYouTuber
- ニュースレター執筆者
- 補完的なツール企業
- コンサルタントやエージェンシー

### アウトリーチテンプレート:
```
件名: パートナーシップの機会 — [プロダクト名]

[名前] さん、

[トピック]に関するあなたのコンテンツ、特に[具体的なコンテンツ]を拝見しており、パートナーシップの可能性があると考えています。

[プロダクト名]は[オーディエンス]が[成果を達成]するのを支援しており、あなたのオーディエンスにも価値を感じていただけると思います。

パートナー向けに[コミッション構造]、さらに[追加特典: 早期アクセス、コマーケティング等]をご用意しています。

詳細をお聞きいただけますか？

[あなたの名前]
```

## ツール・プラットフォーム

### 紹介プログラムツール

**フル機能プラットフォーム:**
- ReferralCandy — EC向け
- Ambassador — エンタープライズ向け紹介プログラム
- Friendbuy — ECとサブスクリプション向け
- GrowSurf — SaaSとテック企業向け
- Mention Me — AIパワードの紹介マーケティング
- Viral Loops — テンプレートベースのキャンペーン

**ビルトインオプション:**
- Stripe（基本的な紹介トラッキング）
- HubSpot（CRM統合）
- Segment（トラッキングと分析）

### アフィリエイトプログラムツール

**アフィリエイトネットワーク:**
- ShareASale — 大規模なマーチャントネットワーク
- Impact — エンタープライズパートナーシップ
- PartnerStack — SaaS向け
- Tapfiliate — シンプルなSaaSアフィリエイトトラッキング
- FirstPromoter — SaaSアフィリエイト管理

**セルフホスト:**
- Rewardful — Stripe統合のアフィリエイト
- Refersion — ECアフィリエイト

### ツールの選び方

考慮事項:
- 決済システムとのインテグレーション
- 不正検出機能
- ペイアウト管理
- レポーティングと分析
- カスタマイズオプション
- コスト vs プログラム規模

## 不正防止

### 一般的な紹介不正
- 自己紹介（偽アカウントの作成）
- 紹介リング（グループでお互いを紹介する）
- クーポンサイトへの紹介コードの投稿
- 偽のメールアドレス
- VPN・デバイスなりすまし

### 防止措置

**技術的:**
- メール認証を必須にする
- デバイスフィンガープリンティング
- IPアドレスモニタリング
- 報酬支払いの遅延（アクティベーション後）
- 最小活動閾値の設定

**ポリシー:**
- 明確な利用規約
- 期間あたりの最大紹介数
- 返金・チャージバック時の報酬回収
- 疑わしいパターンの手動レビュー

**構造的:**
- 紹介されたユーザーに意味のあるアクションを要求する
- 生涯報酬に上限を設ける
- プロダクトクレジットで報酬を支払う（詐欺師には魅力が薄い）
