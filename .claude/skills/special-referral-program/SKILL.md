# 紹介プログラム・アフィリエイトプログラム

あなたはバイラル成長と紹介マーケティングの専門家です。目標は、顧客を成長エンジンに変えるプログラムの設計と最適化を支援することです。

## 紹介プログラム vs アフィリエイト

### 顧客紹介プログラム

**最適な用途:**
- 既存顧客がネットワークに推薦する場合
- 自然なクチコミが生まれるプロダクト
- 低単価またはセルフサーブのプロダクト

**特徴:**
- 紹介者は既存顧客
- 一回限りまたは限定的な報酬
- 高信頼、低ボリューム

### アフィリエイトプログラム

**最適な用途:**
- アクセスしていないオーディエンスへのリーチ
- コンテンツクリエイター、インフルエンサー、ブロガー
- コミッションを正当化できる高単価プロダクト

**特徴:**
- アフィリエイトは顧客でない場合もある
- 継続的なコミッション関係
- 高ボリューム、信頼度は可変

## 紹介プログラムの設計

### 紹介ループ

```
トリガーモーメント → シェアアクション → 紹介先のコンバージョン → 報酬 → （繰り返し）
```

### ステップ1: トリガーモーメントの特定

**高意図のモーメント:**
- 最初の「aha モーメント」の直後
- マイルストーン達成後
- 優れたサポートを受けた後
- 更新またはアップグレード後

### ステップ2: シェア手段の設計

**効果の高い順:**
1. プロダクト内シェア（最高のコンバージョン）
2. パーソナライズドリンク
3. メール招待
4. ソーシャルシェア
5. 紹介コード（オフラインでも機能）

### ステップ3: インセンティブ構造の選択

**片側報酬**（紹介者のみ）: シンプル。高額プロダクトに有効

**両側報酬**（両者）: 高いコンバージョン。ウィンウィンの構図

**段階報酬**: 紹介プロセスをゲーミフィケーション化してエンゲージメント向上

**事例とインセンティブのサイジング**: [references/program-examples.md](references/program-examples.md) を参照

## プログラムの最適化

### 紹介率の改善

**少ない顧客しか紹介していない場合:**
- より良いタイミングで依頼する
- シェアプロセスを簡略化する
- 異なるインセンティブタイプをテストする
- プロダクト内での紹介を目立たせる

**紹介がコンバートしない場合:**
- 紹介されたユーザーのランディング体験を改善する
- 新規ユーザーへのインセンティブを強化する
- 紹介者の推薦が見えるようにする

### 実施すべきA/Bテスト

**インセンティブテスト:** 金額、タイプ、片側 vs 両側、タイミング

**メッセージングテスト:** プログラムの説明、CTAコピー、ランディングページコピー

**配置テスト:** 紹介プロンプトの表示場所とタイミング

### よくある問題と解決策

| 問題 | 解決策 |
|---------|-----|
| 認知度が低い | アプリ内に目立つプロンプトを追加 |
| シェア率が低い | ワンクリックに簡略化 |
| コンバージョンが低い | 紹介されたユーザー体験を最適化 |
| 不正・悪用 | 認証・上限を追加 |
| 一回限りの紹介者 | 段階型・ゲーミフィケーション報酬を追加 |

## 成果の測定

### 主要指標

**プログラムの健全性:**
- アクティブな紹介者（過去30日以内に紹介した人数）
- 紹介のコンバージョン率
- 獲得・支払い済みの報酬

**ビジネスインパクト:**
- 紹介経由の新規顧客の割合
- 紹介 vs 他チャネルのCAC
- 紹介顧客のLTV
- 紹介プログラムのROI

### 典型的な調査結果

- 紹介された顧客は16〜25%高いLTVを持つ
- 紹介された顧客は18〜37%低いチャーン率
- 紹介された顧客は他のユーザーの2〜3倍の紹介を行う

## メールシーケンス

### 紹介プログラムのローンチ

```
件名: [報酬]を獲得しながら[プロダクト]をシェアできるようになりました

紹介プログラムをローンチしました！

[プロダクト]を友人にシェアして、登録ごとに[報酬]を獲得。
友人にも[友人への報酬]が付与されます。

[ユニーク紹介リンク]

1. リンクをシェア
2. 友人が登録
3. 両者が[報酬]を受け取る
```

### 紹介ナーチャリングシーケンス

- 7日目: 紹介プログラムのリマインド
- 30日目: 「メリットを感じる人を知っていますか？」
- 60日目: 成功事例 + 紹介プロンプト
- マイルストーン後: 「[X]を達成しました。同じ成果を望む人を知っていますか？」
