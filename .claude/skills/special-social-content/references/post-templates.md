# 投稿フォーマットテンプレート

異なるプラットフォームとコンテンツタイプ向けのすぐに使えるテンプレート。

## 目次
- LinkedIn 投稿テンプレート（ストーリー投稿、逆張りテイク、リスト投稿、ハウツー）
- Twitter/X スレッドテンプレート（チュートリアルスレッド、ストーリースレッド、分析スレッド）
- Instagram テンプレート（カルーセルフック、リールスクリプト）
- フック公式（好奇心フック、ストーリーフック、バリューフック、逆張りフック、社会的証明フック）

## LinkedIn 投稿テンプレート

### ストーリー投稿
```
[フック：予期しない結果または教訓]

[場面の設定：いつ/どこで起きたか]

[直面した課題]

[試したこと / 何が起きたか]

[転換点]

[結果]

[読者への教訓]

[エンゲージメントを促す質問]
```

### 逆張りテイク
```
[不人気な意見を大胆に述べる]

理由はこちら:

[理由 1]
[理由 2]
[理由 3]

[代わりに推奨するもの]

[議論への招待：「私は間違っていますか？」]
```

### リスト投稿
```
[信頼性の根拠] 後に [トピック] について学んだ [X] つのこと:

1. [ポイント] — [簡単な説明]

2. [ポイント] — [簡単な説明]

3. [ポイント] — [簡単な説明]

[まとめのインサイト]

どれが最も共感できますか？
```

### ハウツー
```
[期間] で [結果を達成する] 方法:

ステップ 1: [アクション]
↳ [なぜこれが重要か]

ステップ 2: [アクション]
↳ [主要な詳細]

ステップ 3: [アクション]
↳ [避けるべきよくあるミス]

[期待できる結果]

[CTA または質問]
```

## Twitter/X スレッドテンプレート

### チュートリアルスレッド
```
ツイート 1: [フック + 価値の約束]

「[結果] する正確な方法（ステップバイステップ）:」

ツイート 2〜7: [詳細を含む 1 ステップ 1 ツイート]

最終ツイート: [まとめ + CTA]

「役立てば、[トピック] についてはフォローしてください」
```

### ストーリースレッド
```
ツイート 1: [興味を引くフック]

「[時間] 前、[予期しないことが起きました]。全話はこちら：」

ツイート 2〜6: [ストーリーのビート、緊張の構築]

ツイート 7: [解決と教訓]

最終ツイート: [テイクアウェイ + エンゲージメントの要求]
```

### 分析スレッド
```
ツイート 1: [会社/人物] がちょうど [何かをした]。

なぜそれが天才的か（そしてあなたが学べること）:

ツイート 2〜6: [分析ポイント]

ツイート 7: [主要なテイクアウェイ]

「[関連するインサイト + フォロー CTA]」
```

## Instagram テンプレート

### カルーセルフック
```
[スライド 1: 大胆な発言または質問]
[スライド 2〜9: スライドごとに 1 ポイント、ビジュアル + テキスト]
[スライド 10: まとめ + CTA]

キャプション: [トピックを拡張し、コンテキストを加え、CTA を含める]
```

### リールスクリプト
```
フック（0〜2 秒）: [パターン割り込みまたは大胆な主張]
セットアップ（2〜5 秒）: [ヒントのコンテキスト]
バリュー（5〜25 秒）: [実際のアドバイス/コンテンツ]
CTA（25〜30 秒）: [フォロー、コメント、シェア、リンク]
```

## フック公式

最初の一行が残りを読むかどうかを決めます。

### 好奇心フック
- 「[一般的な信念] について間違っていました。」
- 「[結果] が起きる本当の理由はあなたの思っているものではありません。」
- 「[印象的な結果] — しかもたった [驚くほど短い時間] で。」
- 「[インサイダーの知識] について誰も語りません。」

### ストーリーフック
- 「先週、[予期しないこと] が起きました。」
- 「危うく [大きなミス/失敗] しそうになりました。」
- 「3 年前、私は [過去の状態]。今日は [現在の状態]。」
- 「[人物] が私に決して忘れられないことを教えてくれました。」

### バリューフック
- 「[一般的な苦痛なし] で [望ましい結果] を得る方法:」
- 「[結果] をもたらす [数字] の [こと]:」
- 「[結果] への最もシンプルな方法:」
- 「[よくあるミス] をやめて、代わりにこうしてください:」

### 逆張りフック
- 「不人気な意見: [大胆な発言]」
- 「[一般的なアドバイス] は間違っています。理由はこちら:」
- 「[一般的な習慣] をやめたら [ポジティブな結果] になりました。」
- 「みんな [X] と言います。真実は [Y] です。」

### 社会的証明フック
- 「私たちは [期間] で [結果を達成しました]。全話はこちら:」
- 「[数字] 人が [トピック] について質問しました。答えはこちら:」
- 「[権威ある人物] が [教訓] を教えてくれました。」
